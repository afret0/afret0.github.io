Posted on 2018-10-25 | Edited on 2019-04-16
手残隐藏了菜单栏解决办法
  • 按住 Ctrl+Shifp+p, 出现一个框，在框里输入 “view:”
  • 选择：“View:Toggle Menu”

安装 package control

sublime 在使用插件前，需要先安装 package control 插件

  • 下载 package control
  • 点击 Preferences->Browser Packages...
  • 返回上级目录，然后进入 Installed Packeges 目录
  • 将刚才下载的文件复制到 Installed Packeges 目录下
  • 重启 sublime

安装 markdown 插件

安装 markmon

  • 点击Preferences->Browser Packages...（或者 Shift + Ctrl + P）
  • 输入 pcip（即选择 Package Control: Install Package）
  • 输入 mmp（即选择 Markmon real-time markdown preview）
  • 回车

安装 markdown preview

  • 点击Preferences->Browser Packages...（或者 Shift + Ctrl + P）
  • 输入 pcip（即选择 Package Control: Install Package）
  • 输入 markdown preveiw
  • 回车

使用 markdown 插件

markdown 语法高亮

  • 点击Preferences->Browser Packages...（或者 Shift + Ctrl + P）
  • 输入 ssm（即选择 Set Syntax: Markdown）

预览 markdown

  • 点击Preferences->Browser Packages...（或者 Shift + Ctrl + P）
  • 输入 mpp（即选择 Markdown Preveiw: preveiw in browser）
  • 选择 git 风格或者 markdown 风格

保存 markdown 为 HTML

  • 点击Preferences->Browser Packages...（或者 Shift + Ctrl + P）
  • 输入 mpsth（即选择 Markdown Preveiw: save to HTML）
  • 选择 git 风格或者 markdown 风格
